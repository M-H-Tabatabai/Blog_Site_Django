Reinforcement learning is a type of machine learning where an agent learns by interacting with an environment and receiving rewards or penalties for its actions.

Machine learning

What is reinforcement learning?

A type of machine learning where an agent learns to make decisions by interacting with an environment — not from labeled examples, but through trial, error, and feedback.

Think of teaching a dog a trick. You don't hand it a manual — you reward good behavior with a treat and discourage wrong ones. Over time it figures out which actions lead to treats. That's reinforcement learning in a nutshell.

Introducing Markov Decision Processes

A Markov Decision Process is the formal mathematical framework used to model sequential decision-making problems — giving reinforcement learning a rigorous structure to work within.

Formally, an MDP is defined by a tuple: (S, A, P, R, γ) — five components that together describe every aspect of a decision problem.

Components

Symbol Name Meaning
S States Set of all possible situations the agent can be in
A Actions Set of all choices available to the agent
P(s'|s,a) Transition Probability of reaching state s' from s after action a
R(s,a,s') Reward Expected reward for transitioning from s to s' via a
γ Discount How much the agent values future rewards (0 ≤ γ ≤ 1)

The gridworld example

Imagine a robot navigating a 4×4 grid, trying to reach a treasure chest while avoiding a pit of fire.

SStart
XWall
FFire
GGoal
S — Start position
X — Obstacle
F — Fire pit  (reward: −10)
G — Goal  (reward: +10)
· — Open cell  (reward: −1 per step)

MDP in this example

📍

States (S)

Each cell position — up to 16 states in the grid

↕️

Actions (A)

Move up, down, left, or right

🎲

Transitions (P)

Deterministic or stochastic — a 10% slip chance makes it harder

⭐

Rewards (R)

−1 each step, −10 for fire, +10 for reaching the goal

⏳

Discount (γ)

Low γ = reach goal fast. High γ = patient long-term planning

Core tension in RL

This simple setup captures it all: exploration vs. exploitation and short-term vs. long-term reward.
